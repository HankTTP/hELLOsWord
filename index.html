JavaScript是什麼?

  • JavaSciprt是一種物件導向程式設計語言、弱型別的 腳本語言.
  • JavaSciprt 不是Java ！ .
  • 兩者差別在於,JavaSciprt是腳本語言, Java是編譯型語言;也就是他們完全沒有關係.
  • JavaSciprt可以用於改善瀏覽器的Client端體驗.
  • JavaSciprt跟眾多程式語言一樣是藝術,請好好欣賞他.

如何使用JavaScript?

  • JavaScript的程式碼可以放再html頁面中,也可存放於外部.
  • 存放於外部的JavaScript則以.js做為副檔名.
  • <script>JavaScript Code <script/> 可放再head或body中.
  • 某Y公司建議可將JavaScript放再body的結束標籤前,此舉可以加快HTML頁面載入呢.
